# Single price grid component

This challenge is to build out the pricing component to the designs provided and get it looking as close to the design as possible. 

Users should be able to:

- View the optimal layout for the component depending on their device's screen size
- See a hover state on desktop for the Sign Up call-to-action

---

## About the project 

This project was created using `npm create vite@latest` with the **React + JavaScript** option.

## How to run this project

To run this project locally, follow these steps:

1) Clone the repository.
`git clone`

2) Navigate to the project folder.
`cd recipe-page`

3) Install dependencies. 
`npm install`

4) Start the development server. 
`npm run dev`

5) Then open the local server link in your browser.
